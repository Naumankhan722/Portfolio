# Portfolio (Next.js starter)

Quick starter for a personal portfolio built with Next.js. Ready for local development and deployment on Vercel.

Getting started

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
npm run start
```

Deploying to Vercel

- Push this repo to GitHub (or connect your Git provider).
- In Vercel, create a new project and import the repo — Vercel auto-detects Next.js.
- Or use the Vercel CLI:

```bash
npm i -g vercel
vercel
```

Files of interest

- `package.json`: scripts and dependencies
- `pages/`: Next.js pages (homepage, about, projects)
